This seed will hopefully be ready to go out of the box and include Token Based security. It is based on the semester seed from the 3rd semester in structure but made as a Webstorm Project. Clone and npm install and it should work !

Features include :

  • JWT Token based authentication
  • Sanitizing of usernames and passwords to prevent script attacks
  • Helmet to protect headers

This is an ongoing project and is not ready for serious use !!
